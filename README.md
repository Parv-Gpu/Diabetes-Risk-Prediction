# Diabetes Risk Prediction

A machine learning project built for diabetes risk prediction using healthcare records. The project focuses on data preprocessing, feature engineering, model training, and evaluation using multiple ML algorithms.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- LightGBM
- Kaggle Notebook

## Project Overview

This project builds an end-to-end machine learning pipeline to predict diabetes risk from healthcare data. The workflow includes cleaning the dataset, engineering clinical and lifestyle-based features, training multiple models, and evaluating performance using classification metrics.

## Dataset

- Source: Kaggle Competition Dataset
- Size: 700K+ healthcare records
- Target: Diabetes risk prediction

## Workflow

1. Data loading and exploration
2. Missing value handling
3. Feature engineering
4. Class imbalance handling
5. Model training
6. Model comparison
7. Performance evaluation

## Feature Engineering

Created domain-inspired features such as:

- BMI categories
- Blood pressure bands
- Cholesterol indicators
- Lifestyle-based risk factors
- Health risk flags

## Models Used

- Logistic Regression
- Random Forest
- XGBoost
- LightGBM

## Evaluation Metrics

- ROC-AUC
- Accuracy
- Precision
- Recall
- F1-Score

## Result

Best ROC-AUC Score: **0.70**

## Key Learnings

- Built a complete ML pipeline from raw data to model evaluation.
- Learned how to handle large-scale healthcare datasets.
- Compared traditional ML models with boosting-based models.
- Improved model robustness using feature selection and tuning.
